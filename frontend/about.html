System Overview

How CarePulse Works

CarePulse calls patients, captures symptoms through IVR, converts speech to structured answers, and updates the risk score. Clinicians review alerts and can confirm, clear, or correct responses to keep the model accurate.

Reviewing Alerts
  • Confirm Alert: The alert is clinically correct. This strengthens the alert signal.
  • Clear Alert: The alert is a false positive (speech error, misunderstanding).
  • Save Correction: Fixes the structured answer if the transcript was misread.
  • All actions are audited: Each review updates the call record and retraining data.
Risk Explainability
  • Model-based: Impact bars explain why the model increased or decreased risk.
  • Manual override: If a doctor overrides risk, the source shows "manual override."
  • Fallback logic: If model explanations are unavailable, rule-based factors are shown.
Actions Tab
  • Override Risk: Sets a new risk score and marks the source as manual override.
  • Mark False Alert: Flags the call as a false positive for training.
  • Retrain Now: Uses the latest clinician feedback to improve the model.
Roles & Access
  • Doctor: Full review, corrections, override, retrain.
  • Nurse: Patient monitoring and alerts without overrides.
  • Staff: Operational monitoring (calls, scheduling).
  • Admin: System health only, no patient-level details.
